Work

  • Project on Government Oversight

    POGO came to us wanting to redesign and replatform their outdated site. As the sole developer on this project, I helped build a custom Craft CMS site from scratch. The new site implements interesting UI patterns and app-like animation. It connects to Algolia to enable robust searching. I also wrote custom PHP plugins to power their workflow.

  • The Nature Conservancy

    Working with another dev, I helped build the Waterfunds Toolbox, a resource library and information platform. This involved complex data modeling in the Craft backend to create an extensible architecture for their large dataset. The Toolbox won a W3 Gold Award in 2018.

  • Optify

    Optify is an online coaching platform that connects clients to leadership professionals. I helped build this Elixir app as the sole frontend developer on a team of three devs. Pairing Elixir with the Phoenix platform, we were able to use websockets for realtime communication and live reloading.

  • Improving the Way We Shop Online

    This exploration on online shopping was an internal marketing site that I built from scratch. It was a good opportunity to try out new technologies —— I experimented with the IntersectionObserver API to monitor when elements come into view, and tried out the Bodymovin library for SVG animations.

  • Bethesda Softworks

    Bethesda wanted to overhaul multiple sites in anticipation for the launch of a new game. I supported this effort on a team with two other devs. Even though we were on a tight deadline, I was able to learn Vue.js in order to implement a custom carousel. I also supported data migration into the Contentful CMS.

  • Craig Hospital

    I overhauled Craig's resource library to freshen up the UI and implement keywords for better search results. This involved learning Craig's existing data architecture so that I could make changes without breaking other parts of the site. I also dove into an old and sprawling codebase and picked up jQuery along the way.

  • White House Historical Association

    I built this one-off marketing page for the White House Historical Association's 2018 Christmas ornament. This involved diving into an existing codebase and leveraging styles to avoid duplicative code.
